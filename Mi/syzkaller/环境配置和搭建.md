- 配置go环境(需要go版本在1.20及以上)

```Bash
wget https://dl.google.com/go/go1.21.4.linux-amd64.tar.gz
tar -xf go1.21.4.linux-amd64.tar.gz
# 配置环境变量
vi ~/.bashrc
# 在~/.bashrc中加入以下两行
export GOROOT=`pwd`/go
export PATH=$GOROOT/bin:$PATH
```

- 下载安装syzkaller（**在压缩包中已经有编译好的bin文件了，这一步可以不用进行**）

```Bash
git clone https://github.com/google/syzkaller
cd syzkaller
make -j1 TARGETOS=linux TARGETARCH=arm64
# 如果报aarch64-linux-gnu-gcc is missing (exec: "aarch64-linux-gnu-gcc": executable file not found in ATH)
# 执行以下命令
sudo apt install gcc-aarch64-linux-gnu
# 如果报aarch64-linux-gnu-gcc: error trying to exec 'cc1plus': execvp: No such file or directory 
# 执行以下命令
sudo apt install g++-aarch64-linux-gnu
```
